# 🛒 Ecommerce Platform

## Project Description

This is a full-stack **Ecommerce Platform** designed to manage and simulate a complete online shopping system.  
The application provides both **user (client)** and **admin** functionalities, covering all essential ecommerce operations.

---

## Features

### User / Client 
- User authentication and account management
- Browse and search products
- View product details
- Add products to cart
- Manage cart (update quantities, remove items)
- Place orders
- Order tracking and delivery status

---

### Admin Side
- Dashboard for system management
- Product management (add, update, delete products)
- Client management
- Order management
- Delivery management

---

## Core Modules
- Product Management
- Client Management
- Order Management
- Delivery Management
- Shopping Cart System
- Authentication & Authorization (Admin / User roles)

---

## Project Goal

The goal of this project is to build a complete ecommerce system that simulates real-world online shopping platforms, including product lifecycle, order processing, and delivery tracking.

---

## Tech Stack
- Backend: FastAPI / Python
- Database: MySQL
- Frontend: Angular (HTML-CSS)
- Authentication: Secure password hashing using bcrypt
